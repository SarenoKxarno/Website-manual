Размеры блока

По умолчанию, размер блока задаётся так, чтобы он был способен вместить контент. Чтобы установить собственные размеры блока, можно использовать 2 свойства: height и width. Чаще всего размер блока указываеться в пикселях, процентах или единицах em.
Пикесли наиболее полпулярны т.к наиболее точно позволяют контролировать размер блока.
При использовании процентов, размер блока задаёться относительно размера окна браузера, а если блок заключён в другой блок, то относительно размера блока.
При использовании em размер блока зависит от размера содержащегося в нём текста.

Ограничения ширины

Некоторые страницы создаются таким образом, чтобы контент менял свой размер в соответсвии с размером экрана.
В этом случае свойство min-width определяет наименьшую ширину блока при уменьшении окна браузера.
А свойство max-width определит наибольшую ширину блока при увеличении.

Перетекание контента

Свойство overflow сообщает браузеру что делать, если объём контента блока превышает его размер. Это свойство может принемать одно из двух значений:

hidden
Данное значение просто скрывает весь контент, который не поместился в блок

scroll
Данное значение добавляет блоку полосу прокрутки, чтобы пользователи могли увидеть дополнительный контент. Свойство overflow особенно удобно, поскольку некоторые браузеры позволяют пользователям настраивать размер шрифта текста. При слишком большом размере страница становится нечитаемой. Сокрытие не помещающегося контента помогает предотвратить наползание элементов друг на друга.

Управлять внешним отступом блока можно с помощью трёх хакарктеристик:

  1. Граница - Каждый блок имеет границы, даже если они не видны или их ширина равна 0 пикселей
  2. Поля - Поля находятся за пределами границ. Можно настроить ширину полей для создания интервала между блоками
  3. Отступы - Отступ это интервал между границей блока и его содержимым

Толщина границ

Свойство border-width используется для управления толщиной границ, данное свойство может принемать значение в пикселях или одно из следующих значений:

  • thin
  • medium
  • thick

Для этого свойства нельзя устанавливать значения в процентах. Вы можете устанавливать размер отдельных границ, используя следующие 4 свойства

  • border-top-width
  • border-right-width
  • border-bottom-width
  • border-left-width

Можно указать разную ширину границ в одном свойстве.
Пример того, как должен выглядеть код: border-width: 2px 1px 1px 1px
В таком случае, значения перечисленные по часовой стрелке: верхняя, правая, нижняя, левая границы

Можно настраивать стиль границ используя свойство border-style оно способно принемать следующие значени:

  • solid - неприрывная линия

  • dotten - пунктирная в виде точек

  • dashed - пунктир

  • double - две сплошные линии

  • groove - врезанная линия

  • ridge - выпуклая линия

  • inset - вдавленная линия

  • outset - выдавленная линия

  • hidden/none - граница не видна

Изменить стиль отдельных границ можно используя следующие свойства

  • border-top-style
  • border-right-style
  • border-bottom-style
  • border-left-style

Отступы

Свойство padding позволяет указать сколько места должно быть между содержимым элемента и его границами. Значения этого свойства чаще всего указывают в пикселях. При использовании процентного соотношения, будет использовано отношение к окну браузера. Важно обратить внимание: если для блока указана ширина, то значение отступа прибавляется к ширине. Можно задать различные значения для каждой стороны блока используя свойства:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Можно также использовать стеногрфическое свойство, где значения перечислятся по часовой стрелке: вверх, вправо, вниз и влево.

Пример того, как должен выглядеть код:padding: 10px 5px 3px 1px

Значения свойства padding не наследуются дочерними элементами таким же образом как значение color свйостваfont-family поэтому нужно указывать величину отступов для каждого элемента, которому они не обходимы.

Закруглённые углы

С помощью свойства border-radius появилась возможность создавать закруглённые углы у блоков. Значение указывает радиус в пикселях. Старые версии браузеров будут отображать блок с прямыми углами.
Можно указать различные указания для каждого угла блока следующим образом:

  • border-top-right-radius
  • border-bottom-right-radius
  • border-bottom-left-radius
  • border-top-left-radius

Также можно использовать стенографические свойства, которые работают по часовой стрелке: верх, правый угод, низ и левый угол.
Пример того, как должен выглядеть код: border-radius: 5px 10px 5px 10px
