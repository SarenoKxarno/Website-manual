Шрифты

font-family - свйоство позволяет указать шрифт, которым должен быть написан текст. Шрифт будет отображаться только в том случае, если он установлен у пользователя. Одновременно , можно указать до 3х шрифтов, чтобы они работали, как альтернатива друг другу, если один из них не установлен у пользователя.

font-size - это свойство указывает размер шрифта. Существует 3 способа того, как это сделать:

  1. Пиксели - позволяет очень точно указать размер шрифта. Выражаеться в "px"
  2. Проценты - по умолчанию размер текста рвен 16px в процентах это 100%. Если изначально задать размер шрифта у body 72% тоесть 12px, а затем задать ещё одно правило для того же тега, которое задёт размер шрифта в 75%, то размер шрифта станет не 12px а 9px(т.к. это будет 75% от 12px)
  3. Единицы ЕМ - это единицы измерения эквивалентные ширине буквы m
Дополнительные шрифты

@font-face - позволяет использовать шрифт даже если он не установлен на ПК пользователя. Очень важно чтобы у загружаемого шрифта была лицинзия которая предусматривает такое.

font-family - это свойтсво пзволяет указать имя шрифта, в качестве значений свойства, для остальной части таблицы стилей.

src - данный атрибут определяет путь к файлу шрифта. Чтобы шрифт точно работал в всех браузерах, можно указать пкти к нескольким версия шрифта.

format - определяет формат в котором поставляется шрифт.

Полужирное начертание

Свойство font-weight позволяет форматировать шрифт текста

  1. normal - использование этого свойства приведёт к тому, что текст будет иметь обычный вес
  2. bold - делает шрифт жирным аналогом тега <b>, но для CSS
Курсив

Свойство font-style - позволяет выделять текст курсивом. Это свойство может принемать 3 значения:

  1. normal - текст имеет обычное начертание
  2. italic - текст становится курсивным
  3. obliqoe - текст становиться наклонным
Пописные и строчные символы

Свойство text-transform используеться для изменения регистра текста. У него есть несколько следующих значений:

  1. uppercase - все символы используют верхний регистр
  2. lowercase - все символы используют нижний регистр
  3. capitalize - каждое слово начинаеться с заглавной буквы
Форматирование текста

Свойство text-decoration принемает следующие значения:

  1. none - это значение сбрасывает любое применине оформления
  2. underline - данное значение добавляет линию под текстом, аналог тега "u" но для CSS
  3. overline - данное значение добавляет линию над текстом
  4. line-trough - это значение создаёт эффект зачёркнутого текста, аналог тега "del" и "s" но для CSS
  5. blink - заставляет текст мигать, но подобное очень осуждается
Межстрочный интервал

Свойство line-height позволяет изменять размер местрочного интервала.
Межстрочный интервал указываеться не в пикселях, а в em. Наиболее хороший размер интервала примерно 1.4-1.5em

Выравнивание

Выравнивание происходит благодаря свойству text-align. Оно модет принемать одно из следующих свойств:

  1. left - определяет что текст должен быть выровнен по левому краю
  2. right - определяет что текст должен быть выровнен по правому краю
  3. center - определяет что текст должен быть выровнен по центру
  4. justify - определяет что каждая строка абзаца, за исключением последней, должна занимать всю ширину текстового поля

Реагирование на действие пользователя

Существует 3 псевдокласса, позволябщих менять цвет элемента в ответ на действия пользователя:

  1. :hover - применяеться когда пользователь наводит на элемент указатель мыши, Данный псевдокласс применяеть для ссылок и кнопок. Но это не работает с сенсорными устройствами т.к. экран не может понять, когда пользователь наводит на него палец
  2. :active - применяеться когда элемент активируеться пользователем, например путём прожатия кнопки или щелчка мышью
  3. :focus - применяется когда элемент имеет фокус. Фокус возникает, когда браузер понимает что вы готовы к взаимодействию. с элементом на странице. Например когда указатель мыши находиться в поле ввода мыши готового к заполнению.
Селекторы атрибутов

При использовании псевдоклассы должны перечисляться в следующей последовательности:

  1. :link
  2. :visited
  3. :hover
  4. :active
  5. :focus

Помимо основных, популярных селекторов, существует и набор атрибутов, позволяющих создавать правила, которые применяются к элементам, имеющим атрибут с конкретным значением

Селектор Пример
[] p[class] - любой элемент <p> с атрибутом <class>
[=] p[class="dog"] - любой элемент <p> с атрибутом <class> значение которого - dog
[~=] p[class~="dog"] - любой элемент <p> с атрибутом <class>, значением которого является список разделённых пробелами слов, среди которых присутствует dog
[^=] p[attr^"d"] - любой элемент <p> с атрибутом, значение которого начинаеться с буквы d
[*=] p[attr*"do"] - любой элемент <p> с атрибутом, значение которого содержит буквы do
[$=] p[attr$"g"] - любой элемент <p> с атрибутом, значение которого заканчиваеться буквой g
