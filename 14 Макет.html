Заголовки и нижние колонтитулы

Элементы: <header> и <footer> могут использоваться в качестве:

  • Основного заголовка или нижнего колонтитула, которые располагаются в верхней или нижней части каждой страницы.
  • Заголовка или нижнего колонтитула отдельного элемента article или section на странице.

Пример работы:

                <header>
                    <h5>Закусочная Боба</h5>
                </header>
                <p>Закусочная Боба - место, где вы можете насладиться вкусной и сытной едой по доступным ценам.</p>
                <footer>
                    ©2023 "Закусочная Боба">
                </footer>
Меню навигации

В элемент <nav> вставляется основное меню навигации сайта

Пример работы:

                <header> 
                    <h5>Закусочная Боба</h5>
                </header>
                <p>Закусочная Боба - место, где вы можете насладиться вкусной и сытной едой по доступным ценам.</p>
                    <nav>
                        <ul>
                            <li>О нас</li>
                            <li>Меню</li>
                            <li>Отзывы</li>
                            <li>Контакты</li>
                        </ul>
                    </nav>
                <footer>
                    © 2023 "Закусочная Боба">
                </footer>
Статьи

Элемент <article> используется для любого независимого раздела страницы.
Это может быть отдельная статья или запись в блоге, комментарий или сообщение на форуме, а также любая другая самостоятельная часть контента.
Если страница срдержит несколько статей, то каждая отдельная статья будет помещена в собственный элемент <article>.
Элементы <article> бывают вложены друг в друга. Например сообщение в блоге может находиться в одном элементе <article> а каждый коментарий к сообщению - в отдельном дочернем элементе <article>.

Пример работы:

                    <article>
                        <figure>
                            <img src="images/бургеры.png" alt="Помещение закусочной">
                            <figcaption>Закусочная</figcaption>
                        </figure>
                        <hgroup>
                            <h2>Сочные бургеры</h2>
                            <h3>Еда быстрого приготовления</h3>
                        </hgroup>
                        <p>
                            Свежие, приготовленные по лучшему рецепту, бургеры. Готовятся провессионалами для любителей вкусно и быстро поесть 
                        </p>
                    </article>
Сторонний контент

Элемент <aside> имеет две цели, в зависимости от того, находится ли он внутри элемента <article> или нет.

◈Когда элемент <aside> расположен внутри элемента <article> он должен содержать информацию, относящуюся к статье, но не обязательно к её общему смыслу.
Например врезка с цитатой или глоссарий может выступать в качестве стороннего содержимого для статьи, к которой они относятся.

◈Когда элемент <aside> расположен за пределами элемента <article> он выступает в качестве контейнера для контента, относящегося ко всей странице.
Например он может содержать ссылки на другие разделы сайта, списко последних сообщений, строку поиска или последние твиты, добавленные автором.

Пример работы:

                    <aside>
                        <section>
                            <h2>Популярные заказы</h2>
                            <a href="">Гамбургер</a>
                            <a href="">Шаурма</a>
                            <a href="">Бутерброды с тунцом</a>
                            <a href="">Картофель фри</a>
                        </section>
                        <section>
                            <h3>Адрес</h3>
                            <p>
                                Закусочная Боба<br>
                                Омск<br>
                                улица Ленина, дом 25
                            </p> 
                        </section>
                    </aside>
Разделы

Элемент <section> группирует связанный контент и обычно каждый раздел имеет собственный заголовок.
Например на главной старнице может быть несколько элементов <section> содержащих такие разделы страниц, как последние новости, самые популярные товары и подписка на рассылку.
Поскольку элемент <section> группирует связанный контент, он может содержать несколько различных элементов <article> объединённых общей темой или целью
Если страница содержит длинную статью, то элемент <section> можно использовать для разбивки статьи на отдельные разделы.
Элемент <section> не следует использовать в качестве оболочки для всей страницы.
Если необходимо использовать большой контейнер, то для решения этой задачи лучше всего будет использовать <div>

Пример работы:

                <section>
                    <h2>Популярные заказы</h2>
                    <a href="">Гамбургер</a>
                    <a href="">Шаурма</a>
                    <a href="">Бутерброды с тунцом</a>
                    <a href="">Картофель фри</a>
                </section>
                <section>
                    <h3>Адрес</h3>
                    <p>
                        Закусочная Боба<br>
                        Омск<br>
                        улица Ленина, дом 25
                    </p> 
                </section>
Группы заголовков

Элемент <hgroup> предназначен для группировки элементов <h1>-<h6> чтобы они рассматривались как один заголовок.
Например элемент <hgroup> может содержать заголовок внутри элемента <h2> и подзаголовок внутри <h3>

Пример работы:

                    <hgroup>
                        <h2>Особые горячие блюда</h2>
                        <h3>Только по самым лучшим ценам</h3>
                    </hgroup>
Рисунки

Элемент <figure> может содержать контент, отосящийся к содержимому статьи.
Важно отметить что статься по-прежнему должна иметь смысл в случае перемещения содержмого элемента <figure> в другую часть страницы или вообще на другую страницу.
По этой причине данный элемент стоит использовать только тогда, когда контент просто ссылкается на элемент, а не тогда когда он является неотёмлимой частью содержимого страницы.
Примеры использования включают:

  1. Изображения
  2. Видео
  3. Графики
  4. Диограммы
  5. Образцы кода
  6. Текст, который потдерживает основной смысл статьи
Элемент <figure> также должен содержать элемент <figcaption> обеспечивающий описание для его контента.

Пример работы:

                    <figure>
                        <img src="images/шаверма.png" alt="шавуха">
                        <figcaption>Сочная шаверма</figcaption>
                    </figure>
Элементы деления

Элемент <div> остаётся важным способом группировки связанного контента, поскольку новые элементы можно использовать только для тех целей, для которых они предназначены.
Когда для группировки контента нет подходящего элемента, можно использовать <div>

Пример работы:

                    <div>
                        <header>
                            <h1>Закусочная Боба</h1>
                            <nav>
                                <!-- nav контент навигации-->
                            </nav>
                        </header>
                        <section>
                            <!-- section контент раздела-->
                        </section>
                        <aside>
                            <!-- aside сторонний контент-->
                        </aside>
                        <footer>
                            <!-- footer подвал страницы-->
                        </footer>
                    </div>
