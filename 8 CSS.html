<link>

Элемент <link> указывает браузеры где лежит файл CSS. Этот элемент использует три следующих атрибута:

  • Атрибут href указывает путь к файлу CSS который, обычно, лежит в папке "css" или "styles"
  • Атрибут type указывает на тип документа, на который указывает ссылка. Должно быть указано значение: "text/css"
  • Атрибут rel определяет отношение между страницей и файлом. При создании сылки на файл, должено быть указано значение "stylesheet"

Селекторы CSS

В данной таблице будут приведны селекторы CSS которые используются для обращения к коду:

Селектор Принцип работы
* {} - Применяется ко всем элементам в документе
h1, h2, h3 {} - Изменения будут проходить во всех перечисленных элементах
.note {} - Изменения будут проходить с элементом, чей атрибут class имеет значение note
p.note {} - Только элементы <p> чьи атрибуты class имеют занчение note
#intro {} - Элемент чей атрибут id имеет значение intro
li>a {} - Элементы <а> являются прямыми потомками элемента li но не все элементы <а> на странице
p a {} - Любые элементы <а> даже когда они распологаються внутри <p>. Даже в том случае, когда между ними распологаются другие элементы.
h1+p {} - Первый элемент <p> который распологается после элемента <h1>, но не любые другие элементы <p>
h1-p {} - Если бы у вас было два элемента <p> являющихся сестринскими по отношению к элементу <h1> то это правило применялось бы к обоим
Принципы каскадирования правил CSS

В CSS есть определённые принципа работы с стилями. Сейчас их рассмотрим.

  • Последнее правило - Если к одному и тому же элементу присваевается какое-то правило, то наибольший вес будет иметь то, которое было последним.
    Например: селектору i было присвоено значение "color: red" но потом, чуть ниже написали новое значение "color: purple". Значит селектор i примет последнее значение, даже если первое всё ещё осталось в коде.
  • Специфичность - Если один селектор более специфичный чем остальные, то он получает приоритет. Тоесть, конкретный селектор более специфичный чем общий, обращение ко всем элементам внутри тега более специфично чем обращение к конкретному элементу, обращение к селектору с id более специфичным чем обращение к конкретному элементу.
  • Важность - Если добавить команду !important после любого свойства, чтобы указать, что его следует считать самым важным из правил.

Некоторые элементы редактирования наследуются, например тип шрифта. Тоесть можно написать ображение к тегу <body> в котором мы меняем шрифты. И эти изменения будут присвоены ко всем элементам в теге <body>

Цвет

color

Ключевым элементом работы с страницей явялеться её цвет. Для работы с цветом используеться селектор color. Выбрать цвет можно несколькими способами:

  • Значение RGB - эти значения выражают цвет в пропорциях красного, зелёного и синего
  • Шестнацатиричный код - код определяющий количество красного, зелёного и синего цвета
  • Имена цветов - существует 147 предопределённых имён цветов, которые распознаються браузерами
  • background-color - это свойство позволяет поменять задний фон блока кода

HSL и HSLA

В CSS3 было добавлено новое свойство hsl. Его используют как альтернативу работы RGB. У этого свойства есть 3 следующих значения.

  • Оттенок - выражается в диапазоне от 0 до 360
  • Насыщенность - выражается в процентах
  • Светлость - принемает процентное значение(0% - белый, 50% - нормальный, 100% - чёрный)

В свойстве hsla альтернативе hsl буква "a" отвечает за значение альфа-каналов, тоесть прозрачности.
Их значение варируется от 0 до 1.0
