Ссылки на другеи сайты и страницы

Ссылки создаются при помощь тега <a>. Но чтобы создать ссылку, которая будт переводить на дургую страницу, нужно использовать атрибут <href>.
Чтобы ссылка вела на другую страницу нужно использовать следующее оформление: href="1 Работа с html.html".
Если вы хотите чтобы ссылка вела на другую страницу в интернете, то следует использовать следующее оформление: href="https://animego.org/anime/season/2013".

Ссылки на электронную почту

Для создания ссылки, которая запускала бы программу электронной почты, установленную на компьютере пользователя, и позволяла бы отправлять электронные письма на указанный вами адрес, вам нужно будет использовать элемент <a>. Однако теперь значение атрибута href должно начинаться со слова mailto: после которого необходимо будет указать адрес электронной почты, на который посетитель должен будет отправить письмо.

Пример того, как должен выглядеть код: a href="mailto:gonki2001@mail.ru"

Работа с изображениями

<img>

Для добавления изображения на страницу нужно использовать элемент <img>.

src

Данный атрибут сообщает браузеру расположение нужного изображения. Как правило, его значением будет относительный URL-адрес.

alt

Данный атрибут предоставляет текстовое описание изображения, выводимое на экран в случае, если нет возможно вывести на экран само изображение.

title

С элементом <img> также можно использовать атрибут title для сообщения дополнительной информации об изображении.

Пример работы:

Пример того, как должен выглядеть код: <img src="путь изображения" alt="текстовое описание изображения">

Высота и ширина изображений

height

Данный атрибут устанавливает высоту изображения в пикселях.

width

Данный атрибут устанавливает ширину изображения в пикселях.

Пример работы:

Пример того, как должен выглядеть код: <img src="путь изображения" alt="текстовое описание изображения" width="600" height="450">

иллюстрации и подписи

<figure>

Элемент <figure> объединяет изображение и подпись к нему, таким образом текст и рисунок оказываются связанными друг с другом.

<figcaption>

Элемент <figcaption> позволяет добавлять подписи к публикуемым изображениям.
