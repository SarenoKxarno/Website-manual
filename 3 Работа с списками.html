Упорядоченные списки

<ol>

Упорядоченные списки создаются при помощи элемента <ol>

<li>

Каждый элемент списка помещается между тегами <li>

Пример работы:

  1. В этих списках используеться нумерация
  2. Браузер автоматически расставляет нумерацию
  3. Обычно в спиках используется атрибут <type> который применяется в CSS
  4. В CSS можно изменять размер, тип нумерации или вовсе уберать нумерацию и обозначения в списках
  5. Лучше всего использовать язык SCSS т.к. в нём проще редактировать большие объёмы информации
Неупорядоченные списки

<ul>

Неупорядоченные списки создаются при помощи тега <ul>

<li>

Каждый элемент списка помещается между тегами <li>

Пример работы:

  • Это неупорядоченные списки
  • Их можно использовать в качестве простого перечисления функций или ведения списка дел
  • Как и список созданный при помощи атрибута <li> этот список можно редактировать в CSS

Списки определений

<dl>

Список определений создаётся с помощью тега <dl> и, как правило содержит набор понятий и их определений.
Внутри элемента <dl> зачастую можно увидеть пары элементов <dt> и <dd>

  • dt - используется для обозначения понятия(термин)
  • dd - используется для определения понятия

Пример работы:

Биорезонанс
Это концепция, основанная на предположении о том, что каждый орган или система организма имеет свою собственную уникальную энергетическую частоту. С помощью специальных устройств и техник, биорезонанс используется для измерения и коррекции этих энергетических частот с целью восстановления баланса и благополучия организма.
