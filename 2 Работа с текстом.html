Заголовки

В языке HTML существует шесть уровней заголовов:
<h1> - используется для основных заголовков.
С <h2> по <h6> - для подзаголовков.

Если текст разделён на фрагменты более глубокого уровня, то для них используются заголовки <h3> и ниже

Пример работы:

Основной заголовок

Подзаголовок 2го уровня

Подзаголовок 3го уровня

Подзаголовок 4го уровня

Подзаголовок 5го уровня
Подзаголовок 6го уровня
Абзацы

Для создания абзаца необходимо заключить текст в тег <p>
По умолчанию браузер автоматически отображает каждый абзац к новой строки.

Пример работы:

➤ Шоколад из Швейцарии считается одним из лучших в мире благодаря своему непревзойденному качеству и неповторимому вкусу. Шоколадные мастера Швейцарии используют только самые лучшие ингредиенты, такие как натуральное какао, свежее молоко и натуральные ароматизаторы, чтобы создавать шоколадные деликатесы, которые покоряют сердца гурманов по всему миру.

➤ Богатый и насыщенный вкус, гладкость и нежность ломтиков шоколада, раскрывающиеся на языке, делают швейцарский шоколад поистине незабываемым. Если вы еще не пробовали шоколад из Швейцарии, то вам обязательно стоит попробовать его и насладиться этим неповторимым вкусом.

Полужирное и курсивное начертание

<b>

Чтобы выделить текст жирным начертанием шрифта нужно поместить его в теги <b>
Элемент <b> также обозначает текст, внешний вид которого должен отличаться от основного текста абзаца

Пример работы:

➤Каштаны - это вкусный и полезный продукт, который можно употреблять как в сыром, так и в готовом виде. Каштаны богаты крахмалом, клетчаткой, витаминами и минералами, такими как магний, калий и железо. Они могут быть использованы для приготовления различных блюд, таких как супы, пюре, запеканки и кондитерские изделия, а также заменять муку в некоторых рецептах. Каштаны также являются популярным ингредиентом в итальянской и французской кухне.

<i>

Чтобы отобразить текст курсивным начертанием шрифта, его следует поместить между тегами <i>
Элемент <i> определяет фрагмент текста, отличающийся от основного, например технологические термины, иностранные слова и прочие виды заключений в текст, которые принято выделять курсивом.

Пример работы:

➤При проектировании нового компьютера инженеры уделяют большое внимание оптимизации процессора и увеличению его тактовой частоты. Однако, для достижения максимальной производительности, также необходимо использовать быструю оперативную память и SSD-накопители с высокой скоростью чтения и записи данных. Это позволяет сократить время загрузки операционной системы, уменьшить время отклика приложений и повысить общую производительность компьютера.

Подстрочные и надстрочные знаки

<sup>

Элемент <sup> используется для обозначения символов, которые должны быть отображены как надстрочные, например математические понятия.

Пример работы:

➤Один из примеров математического выражения с использованием надстрочных символов - это формула для расчета квадрата числа:
a2 = a × a
Здесь "2" - это надстрочный символ, обозначающий возведение числа в квадрат.

<sub>

Элемент <sub> используется для отображения символов как подстрочных.
Подстрочные символы, как правило, используеются в химических формулах.

Пример работы:

➤Один из примеров химического выражения с использованием подстрочных символов - это формула для воды:
H2O
Здесь "2" - это подстрочный символ, обозначающий количество атомов водорода в молекуле воды.

Важная информация и логические ударения

<strong>

Элемент <strong> используется для обозначения тогоЮ что текст опмещённый в него имеет высокий уровень важности.
Например текст выделленный этим тегом, может быть интонационно выделен программой экранного доступа.
По умолчанию, браузеры выделяют текст в теге <strong> жирным шрифтом

Пример работы:

➤Не забывайте надевать защитные очки при работе с инструментами, которые могут нанести ущерб вашим глазам.

<em>

Элемент <em> используется для обозначения логического ударения, которое несколько изменяет занчение всего предложения.
По умолчанию браузеры отображают содержиоме элемента <em> шрифтом с курсивным начертанием:

Пример работы:

  • Я думаю, Анна была первой
  • Я думаю, Анна была первой
  • Я думаю, Анна была первой
Цитаты

<blockquote>

Элемент <blockquote> используется для отображения длинных цитат, занимающих целый абзац.
Внутри элемента <blockquote> используется и элемент <p>
В большинстве своём, браузеры склонны делать отступы выделяя элемент <blockquote>

Пример работы:

"Жизнь как дикий океан - нужно иметь смелость противостоять ему, чтобы обнаружить его самые великие сокровища!"

<q>

Элемент <q> принято использовать для коротких цитат, помещающихся внутри абзаца текста.

Пример работы:

Когда-то один пьяный ирландец сказал мне перед боем: Хорошая драка - это как хорошее пиво, никогда не бывает слишком много!

Источники и определения

<cite>

Элемент <cite> может быть использован для оформления ссылки на какой-либо первоисточник, например на книгу, кинофильм или исследовательскую работу, для обозначения упоминающегося источника.

Пример работы:

В мире была очень популярна книга Путишествие на край Галлактики, особенно после её успешной экранизации

<dfn>

Элемент <dfn> используется для обозначения какого-либо нового термина.

Пример работы:

Энтропия - В информатике это мера неопределенности или неизвестности в системе передачи информации.

Аббревиатуры и акронимы

При использовании в тексте каких либо акронимов или аббревиатур следует использовать тег <abbr>
В открывающем теге можно указать атрибут <title> для указания расшифровки сокращения.
Для расшифровки акронимов следует использовать тег <acronym>. Для расшифроки которого нужно указывать атрибут <title>.

Пример работы:

Пр. Стивен Хоккинг

NASA

Информацио о веб-дизайнере

Элемент <address> имеет специфическое назначение: он содеожит контактные данные создателя веб-страницы.
Он может содержать адрес электронной почты или номер телефона.

Пример работы:

SKxarn@yandex.ru

г. Санкт-петербуг ул. Ленина, дом 25, квартира 7

Исправления контента

<ins> и <del>

Элемент <ins> может быть использован для выделения контента, добавленного в документ, в то время как элемент <del> - для отображения удалённого контента.

Пример работы:

Это была худшая лучшая поездка за город.

<s>

Элемент <s> позволяет указать что какая-то информация устарела или она ошибочна, но удалять её со страницы не стоит.

Пример работы:

Цена на услуги слесоря: 23000 рублей
