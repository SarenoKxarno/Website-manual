На этой странице представлена работа с таблицами

Базовая структура таблицы

<table>

Для создания таблиц используеться тег <table> записи в таблице осущетсвляются построчно.

<tr>

Открывающий тег <tr> Обозначает начало строки таблицы. После закрытия тега, строка в таблице заканчивается.

<td>

Каждая ячейка таблицы создаётся при помощи тега <td> внутри которого пишится нужная информация. Некоторые браузеры могут оформлять таблици.
Но рекомендуеться делать это самостоятельно в CSS или SCSS

Пример работы:

Макро игра

1 Тибо 2 SarenKxarn 3 Влад

Микро игра

1 Limon 2 Тибо 3 SarenKxarn
Заголовки таблиц

<th>

Можно использовать элемент <th> который используется для создания заголовок стобца или строчки. Даже если ячейка не содержит данных, она должна представать пустым элементом.
Ещё можно использовать атрибут <scope> для обозначения того, является этот элемент строчкой или столбцом.
Для этого используют 2 значения <row> если это строчка и <col> если это столбец.

Пример работы:

Роли Пул
SarenKxarn: 4 15+
Limon: 2 5+
Объединение столбцов

<colspan>

Используя атрибут <colspan> можно задать количество столбцов, которое занимает указанная ячейка.
Этот атибут работает в паре с элементами <td> и <th>

Пример работы:

Голда Мид Экспа
Люнокс Стоит Нестоит
Госсен Нестоит Стоит

А применяя атрибут <rowspan> можно сделать так, чтобы 1 ячейка занимала сразу 2 столбца

Зарядка Пробежка
Понедельник Будет Не будет
Вторник Будет
Длинные таблицы

У таблиц должно быть своё, особенное, оформление. Лучше всего его писать непосредственно в "CSS" но если такой возможности нет, то нужно использовать следующие команды.
<thead> - это тег в котором должен быть заголовки таблицы.
<tbody> - содержит в себе само тело таблицы.
<tfoot> - последняя, завершающая строка таблицы.
Некоторые браузеры могут не отображать вышеупомянутые теги, поэтому вебдизайнеры работают с ними через "CSS" но использовать их стоит для упрощения загрузки браузером очень большой таблицы.

Старый код для редактирования таблиц

Ранее, до того как все визуальные работы начали проводить в CSS для редактирования ширины таблицы использовали атрибут <width> он устанавливал ширигу, которой следовали все остальные элементы строки.
Был и атрибут <cellspacing> задающий величину промежутками между ячейками таблицы.
Атрибут <border> используеться с элементами <table> и <th> для установки ширины границы в пикселях(создания рамки).
Атрибут <bgcolot> используеться для установки цвета всей таблицы.
