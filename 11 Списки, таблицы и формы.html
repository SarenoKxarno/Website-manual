Маркеры

Свойство list-style-type позволяет выбрать форму или стиль маркера. Данное свойство может применяться в правилах, применяемых к элементам ol, ul и li

Неупорядоченные списки

Для неупорядоченных списков можно использовать следующие значения:

  • none
  • disc
  • circle
  • square

Упорядоченные списки

Для упорядоченных списков можно использовать следующие значения:

decimal:
  1. A
  2. B
  3. C
decimal-leading-zero:
  1. A
  2. B
  3. C
lower-alpha:
  1. X
  2. Y
  3. G
upper-alpha:
  1. X
  2. Y
  3. G
lower-roman:
  1. A
  2. B
  3. C
upper-roman:
  1. A
  2. B
  3. C
Маркеры изображения

С помощью свойства list-style-image можно указать в качестве маркера изображение.
Значение начинается с фрагмента URL за которым следует пара скобок. Внутри скобок, в двойных кавычках указывается путь к изображению.
Данное свойство может использоваться в правилах применяемых к элементам ul и li

Позиционирование маркеров

Для списков по умолчанию создаются отступы, а свойство list-style-position указывает должен ли маркер распологаться внутри или снаружи блока, содержащего элементы списка.
Данное свойство может принемать одно из двух значений:

  • outside - При этом маркер распологается слева от текстового блока
  • inside - Маркер распологается внутри блока с текстом, для которого был создан отступ

Свойства таблицы

Здесь собраны все часто используемыме свойства таблиц:

  1. width - устанавливает ширину таблицы
  2. padding - определяет расстояние между границей каждой ячейки и её содержимым
  3. text-transform - используется для переключения регистра заголовков таблицы на верхний
  4. letter-spacing, font-size - добавляет дополнительные стили к заголовкам таблицы
  5. border-top, border-bottom - добавляет границы над и под заголовками таблицы
  6. text-align - выравнивает тект по левому и правому краю
  7. bacground-color - заменяет фоновый цвет чередующихся строк таблицы
  8. :hover - выделяет строку таблицы при наведении на неё указателя мыши

Ниже приведы советы, которые позволяют улучшить восприятие таблиц:

Создавайте отступы в ячейках

Если текст в яейках таблиц касается границ(или другой ячейки), то таблицу сложнее воспринемать. Добавление отступа увеличивает удобочитаемость

Выделяйте загаловки

Выделение заголовков полужирным начертанием шрифта(стиль по умолчанию для элементов th) улучшает читабельность. Вы также можете использовать верхний регистр, для текста заголовков и добавить фоновый цвет или подчёркивание

Оттеняйте чередующие строки

Назначение оттенка каждой второй строке способно помочь пользователям, следовать по строкам. Используйте небольшое отличие от обычного цвета строк талицы, чтобы она выглядела опрятно

Выравнивание номера

Можно использовать свойство text-align, чтобы выровнять содержимое каждого столбца, содержащего номера, по правому краю так, чтобы крупные цифры отличались от юолее мелких

Границы пустых ячеек

Если в таблице есть пустые ячейки, то можно использовать свойство empty-cells, чтобы указать, следует ли отображатьих границы.
Поскольку браузеры по разному обращаются с пустыми ячейками, если нужно явно указать, что границы должны быть показаны или скрыты, то следует использовать данное свойство. Оно можетпринемать одно из трёх значений:

  1. show - при использовании этого значения, границы пустых ячеек отображаются
  2. hide - при использовании этого значения, границы пустых ячеек скрываются
  3. inherit - если одно граница вложена в другую, то данно езначение приводит к тому, что к ячейкам вложенной таблицы применяются правила содержащей их таблицы

В примере видно, что границы пустой ячейки в первой таблице отображаются а во второй - нет.

Интервалы между ячейками

Свойство border-spacing позволяет настраивать расстояние между соседними ячейками. По умолчанию браузеры обычно оставляют небольшой интервал между ячейками таблицы, и, чтобы увеличить или уменьшить его, вы можете использовать свойство, border-spacing.
Значение этого свйоства обычно задаётся в пикселях. Можно указать 2 значения, если нужно отдельно настроить Горизонтальные и Вертикальные интервалы.
Если для ячеек таблицы задана граница, то в местах соприкосновения линия будет в два раза толще, чем по краям таблицы. Чтобы предотвратить это, следует испольщовать свойство border-collapse у него может быть 2 значения:

  1. collapse - При использовании данного значения границы сливаются в одну
  2. separate - Границы отделятся друг от друга

Стилизация кнопок подтверждения

Здесь будут приведены свойства, которые можно использовать для применения стилей к кнопкам подтверждения

  1. color - используется для изменения цвета текста на кнопке

  5. :hover - использован для изменения внешнего вида кнопки, при наведении на неё указателя мыши

Стилизация элементов Fieldset и legend

Элемент fieldset особенно полезен при определении границ формы. В случае с длинной формой, с помощью него можно группировать связанные данные.
Элемент legend используется для добавления группе элементов информации.
Обычно с этими двумя элементами использутся следующие свойства:

  1. width - используется для настройки ширины полей
  2. color - используется для изменения цвета текста
  3. background-color - используется для изменения цвета позади элемента
  4. border - используется для настройки внешнего вида границы вокруг группы элементов и/или надписи
  5. border-radius - используется для создания закруглённых углов в браузерах, которые это поддерживают
  6. padding - может использоваться для добавления пространства внутри элементов
